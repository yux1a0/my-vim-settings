# Vim-Plugin

VIM的插件和主题

#### 1. install Vundle
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
#### 2. 复制vimrc配置文件
```
cat vimrc > ~/.vimrc
```
#### 3. 更新插件
打开`vim`，进入`NORMAL`状态，键入`: PluginInstall`。

#### 4. 创建文件夹
```
mkdir ~/.vim/vimundo/ ~/.vim/vimbackup ~/.vim/vimswap
```

#### 安装好啦，可以愉快的写代码啦！^_~~
